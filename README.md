# moviesapp
Небольшой пет-проект. Беру фильмы с omdb api. Использую vuex. Вывожу небольшую информацию с помощью bootstrap.

## Project setup
```
npm install
```
### Compiles and hot-reloads for development
```
npm run serve
```
